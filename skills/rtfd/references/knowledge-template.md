# Knowledge Mode Template

Use this template when the input is articles, notes, URLs, transcripts, or meeting recordings.

```markdown
---
title: "Article Title"
description: "One-line summary"
author: agent
created: YYYY-MM-DD
modified: YYYY-MM-DD
tags: [relevant, tags]
category: research
---

# Article Title

## Overview
What this is about and why it matters. 2-3 sentences max.

## Key Takeaways
- Takeaway 1
- Takeaway 2
- Takeaway 3

## [Topic-specific sections]
Details organized by theme, not by source document structure.
Distill, don't transcribe.

## Related
- [[wiki-link-1]]
- [[wiki-link-2]]
```

## Guidelines

- Lead with takeaways — the reader wants conclusions first
- Bullet points over paragraphs
- If the source spans multiple topics, create separate articles and cross-link
- Preserve source attribution (URL, author, date) in frontmatter or a Sources section
- One article per concept, not per source document
